## cnpm

- [cnpm/cnpm](https://github.com/cnpm/cnpm)
    - cnpm: npm client for China mirror of npm https://npm.taobao.org

```
$ npm install cnpm -g --registry=https://registry.npm.taobao.org
```

## gulp

```
$ cnpm install gulp-cli -g
$ cnpm install gulp -D
$ touch gulpfile.js
$ gulp --help
```
