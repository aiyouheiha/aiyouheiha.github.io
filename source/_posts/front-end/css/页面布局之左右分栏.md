## Flex 布局

```
<!DOCTYPE html>
<html lang="zh">
<head>
  <meta charset="UTF-8">
  <title>页面布局之左右分栏</title>
  <style>
  .container {
    display: flex;
  }
  .left {
    width: 300px;
    border: 2px solid yellow;
    background: orange;
  }
  .right {
    flex: 1;
    border: 2px solid black;
    background: grey;
  }
  </style>
</head>
<body>
<div class="container">
  <div class="left">
    LEFT
    <ul>
      <li>1</li>
      <li>2</li>
      <li>3</li>
      <li>4</li>
    </ul>
  </div>
  <div class="right">RIGHT</div>
</div>
</body>
</html>
```
