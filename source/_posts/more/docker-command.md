- `docker version`
    - 显示 Docker 版本信息

- `docker search redis`
    - 从 Docker Hub 查找镜像

- `docker pull redis` or `docker pull redis:latest` or `docker pull redis:other-version`
    - 从镜像仓库中拉取或者更新指定镜像

- `docker images` or `docker images redis`
    - 列出本地镜像
