之后可以发现 6381 对应的配置文件已经被重写（6380 与 6382 不会）

重启服务，会发现验证也仅在 6381 节点上有效，因此

  • 如果想要对部分 slot 设置密码，在对应节点上进行配置（或者直接修改配置文件）
  • 全部设置密码，则需要在每个节点上进行操作（直接修改配置文件会更方便些）
